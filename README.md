# todo-list （frontend）✅
山东大学崇新学堂 | 开放性创新实践Ⅰ软件任务 前端代码

🧑‍💻Author： Rui_Yang

🖥️这是一个基于SpringBoot框架和React —— AntD前端组件开发的todolist待办软件

⚙️基本功能包括：
新建任务、编辑任务、删除任务、标记完成、
今日待办、任务查询等

番茄钟🍅 AI任务建议助手🤖

数据库：MySQL
